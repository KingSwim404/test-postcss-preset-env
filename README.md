###  重点配置：

 importFrom:一定不能没有。需要你手动指定你定义的CSS路径。

```json
"postcss": {
    "plugins": {
      "postcss-preset-env": {
        "stage": 1,
        "importFrom": "src/assets/common.pcss"
      }
    }
  }
```
